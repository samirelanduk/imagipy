imagipy
=======

imagipy is an image and color processing library.

Example
-------

  >>> import imagipy
  >>> red = Color(255, 0, 0)
  >>> green = Color.from_hex("#00FF00")
  >>> blue = Color.from_hex("0000FF")
  >>> red.mutate()
  <Color (249, 2, 7)>
